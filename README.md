# DoNotPlay
DoNotPlay 是一款基于人工智能视觉的桌面工具，它通过监控疲劳与分心状态，帮你用高效的工作换取更多娱乐时间。An AI-powered desktop tool that prevents distractions by monitoring your posture and fatigue, helping you trade efficient work for more playtime.


🛡️ DoNotPlay - 你的专属 AI 专注力卫士
“为了毫无负罪感地尽情玩耍，请在此刻保持绝对专注。”

你是否也经常深受“注意力涣散”的困扰？明明打算工作或学习，却总是不由自主地拿起手机？DoNotPlay 就是为你量身打造的桌面级 AI 技术工具。

💡 核心理念：为什么叫 DoNotPlay？
时间对每个人都是绝对公平和恒定的。如果你在工作时间不停地摸鱼，实际上是在透支你本该用来休息和娱乐的时间。
虽然这个项目名叫 DoNotPlay（不要玩），但它的终极本意其实是 Play More（玩得更多）—— 帮助你在该专注的时候极度专注，从而为你省下更多的时间去尽情地、毫无负担地享受生活。

✨ 核心功能
本程序通过纯本地的计算机视觉技术（不上传任何隐私数据），将你的摄像头变成一位极其严格的“监考老师”：

👁️ 多维度视觉追踪：实时捕捉你的眼部视线（Gaze）、手部动作（Hands）以及头部姿态（Head Pose）。

📱 手机与分心监测：精准判断你是否在工作期间玩手机或左顾右盼。

💤 疲劳程度评估：通过监测眨眼频率（PERCLOS）和打哈欠、点头等动作，科学判断你的疲劳状态，并在你需要休息时发出提醒。

🎯 智能情景过滤：算法经过专门调优，尽可能降低了正常工作行为（如低头看书、敲击键盘、使用鼠标）带来的误判干扰。

# 🛡️ DoNotPlay 快速使用指南

欢迎使用 DoNotPlay！这是一款纯本地运行的 AI 专注力卫士。以下是详细的安装与使用步骤。

## 1. 环境准备与安装

由于本程序依赖人工智能视觉模型，首次运行前需要配置 Python 环境。

**步骤一：确保已安装 Python**
请确保你的电脑上已安装 Python 3.8 或更高版本。

**步骤二：安装依赖库**
打开命令行终端（Windows 用户可按 `Win + R` 输入 `cmd`），进入你下载的 DoNotPlay 文件夹目录，然后运行以下命令一键安装所需环境：

```bash
pip install -r requirements.txt
```
*(注：初次安装可能需要几分钟时间，系统会自动下载 YOLOv8 和 MediaPipe 视觉模型，请耐心等待。)*

---

## 2. 启动程序

在命令行终端中，确保当前路径为项目根目录，输入以下命令启动程序：

```bash
python server.py
```

* **默认模式**：程序会自动弹出一个原生的桌面软件窗口。
* **开发/网页模式**：如果你想在浏览器中打开它，可以加上 `--dev` 后缀，即运行 `python server.py --dev`。

---

## 3. 核心使用流程

程序启动后，请按照以下步骤进入专注状态：

1.  **允许摄像头访问**：程序需要调用你的前置摄像头。请放心，所有画面都在本地处理，**绝不上传网络**。
2.  **正坐并校准 (关键)**：
    * 在座位上保持你平常工作时最舒适、端正的坐姿。
    * 直视屏幕，然后点击界面右上角的 **“校准”** 按钮（或按键盘 `C` 键）。
    * 保持不动 3 秒钟。程序会记录你当前的头部高度和眼睛位置，作为后续判定驼背和分心的基准。
3.  **调整灵敏度**：
    * 点击 **“灵敏度”** 按钮（或按 `S` 键）可以在 `Tight (严格)`、`Normal (正常)`、`Wide (宽容)` 和 `Very Wide (极宽容)` 之间切换。
    * *建议*：如果你只有一块屏幕，使用 `Normal`；如果你有两到三块屏幕需要经常转头，请切换到 `Wide`。
4.  **开启番茄钟**：
    * 点击下方的 **“开始专注”**，番茄钟开始倒计时，立刻进入深度工作状态！

---

## 4. 警报机制与应对说明

DoNotPlay 会化身最严格的监考老师，当你出现以下行为时，它会触发全屏弹窗警告：

* 📱 **手机警报 (DoNotPlay!)**：一旦画面中出现手机，并且你正在注视它，屏幕会立刻闪烁红色警告。**应对**：立刻放下手机，点击“我已放下手机”按钮。
* 🦴 **驼背/姿势警报**：当你的面部高度明显低于校准时的基准线，且持续超过几秒钟时触发。**应对**：挺直腰背，点击“我已坐直”。
* 💤 **疲劳警报**：系统综合计算你的眨眼频率（闭眼占比过高）、打哈欠和点头动作，判定你极度疲劳。**应对**：建议起身喝水或闭目养神，点击“我已清醒”。
* ⚠️ **分心警报**：检测到你的视线长时间偏离屏幕（左顾右盼）。**应对**：将注意力收回屏幕，点击“我已恢复专注”。
* 😭 **离开预警**：摄像头长时间捕捉不到你的人脸。**应对**：回到座位前。

---

## 5. 快捷键速查表 (全局可用)

为了让你无需频繁使用鼠标即可控制程序，我们提供了丰富的快捷键：

| 快捷键 | 功能 | 说明 |
| :--- | :--- | :--- |
| **`C`** | 校准姿势 | 重新记录面部中心基准点 |
| **`S`** | 切换灵敏度 | 根据你的显示器数量调整防误判范围 |
| **`P`** | 番茄钟启停 | 快速开始或暂停当前的番茄钟 |
| **`N`** | 跳过当前阶段 | 直接进入下一个休息或专注循环 |
| **`空格键`** | 全局暂停 | 临时离开时暂停所有监测与计时 |
| **`M`** | 静音切换 | 开启或关闭所有提示音和警告音 |
| **`T`** | 切换主题 | 在深色模式与浅色模式间切换 |
| **`Q`** | 会话摘要 | 查看本次专注的综合数据报告 |
| **`E`** | 导出数据 | 导出当前会话数据并关闭摘要页面 |

---

**💡 小贴士**：你可以随时在右侧的仪表盘中查看自己的瞬目率、肩颈平衡状态以及连续专注时长。将每天的专注时间作为游戏打怪升级，祝你每天都能达成 100% 的专注目标！
